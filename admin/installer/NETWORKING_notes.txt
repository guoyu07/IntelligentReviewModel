To see your DNS server address type following command as shell prompt:
cat /etc/resolv.conf

nameserver 143.215.249.152
nameserver 128.61.244.253
nameserver 130.207.244.81

Networking:
cat /etc/resolv.conf

sudo vim /etc/network/interfaces

dns-nameservers 143.215.249.152

sudo /etc/init.d/networking restart
